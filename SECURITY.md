# Security Policy

## Supported versions

Only the latest release of FlashCard receives security fixes.

| Version | Supported |
|---|---|
| Latest release | ✅ |
| Older versions | ❌ |

## Reporting a vulnerability

Please **do not open a public issue** for security vulnerabilities.

Report them privately through GitHub:
[Report a vulnerability](https://github.com/allisterk2703/flashcard/security/advisories/new)

You can expect an acknowledgment within a few days. Once the issue is
confirmed and fixed, a new release is published and the advisory is disclosed.

## Scope

FlashCard is a local-only macOS app: it stores data on your machine and its
only network access is the update check against the GitHub API. Reports about
dependencies with known CVEs are welcome, but exploitability in this local
context will be assessed case by case.
